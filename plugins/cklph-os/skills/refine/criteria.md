# Refine — convergence rubric

Lazy-loaded depth for the `refine` phase. Use to converge on a direction once you've diverged.

## Is it worth building?
- **Painkiller vs vitamin** — does it remove real, felt pain, or is it nice-to-have? Painkillers win.
- **Value × Feasibility** — high-value + feasible → do now; high-value + hard → de-risk first;
  low-value → drop, however fun.

## Tier the assumptions
- **Must be true** (the idea dies if it's false) → validate *first*.
- **Should be true** → check soon.
- **Might be true** → note and proceed.

## Converge
Pick **one** direction. State: the bet, what would kill it, and what you're deliberately **not** doing.

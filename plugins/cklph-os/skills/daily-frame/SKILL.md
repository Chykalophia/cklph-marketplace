# daily-frame — bookend the day

Pick a mode based on time of day, or whichever the user names.

## Morning frame
- **One focus today** — the single thing that makes today a win. Not three. One.
- **What's at risk** — the meeting / dependency / energy dip that could derail the focus.
- **What a win looks like** — the concrete artifact or state at end-of-day. ("Spec merged" beats "made progress.")

## Evening shutdown
- **What shipped** — the concrete output (PR / commit / sent email / decision made). Real verbs only.
- **What didn't (and why)** — the planned thing that didn't land, plus the *cause* (not the excuse). Pattern detection beats self-flagellation.
- **Tomorrow's one focus** — set the morning's focus tonight, while context is fresh.

## Anti-pattern
**Journaling.** This is a frame, not a feelings-log. Every line is action-shaped: a noun + a verb + a thing. If you find yourself writing "I felt overwhelmed because…", stop — that's a different tool. Daily-frame is for *what* and *what next*.

Keep both modes ~6 lines total. Discipline is the point.

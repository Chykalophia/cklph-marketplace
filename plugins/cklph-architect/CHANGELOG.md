# Changelog

All notable changes to `cklph-architect` are documented here.
Format: [Keep a Changelog](https://keepachangelog.com); versioning follows [SemVer](https://semver.org).

## [0.1.0] — 2026-05-31

### Added — initial release
- **`discover`** — diagnostic conversation skill: outcome / impact / constraints / decision-maker before any prescription.
- **`scope`** — outcome-not-deliverables scoping rules; define "done" by business result, not by hours.
- **`propose`** — options-not-estimates pattern (2-3 tiered options, never a single number).
- **`price`** — value-anchored pricing logic; cost-plus is the floor, value is the ceiling.
- **`position`** — peer-not-vendor framing, the lead-the-conversation pattern, and the decline-well pattern.
- **`engagement-shape`** — selecting between project / retainer / advisor / fractional patterns by client type and value pattern.

### IP discipline
This plugin codifies *operating discipline* in Peter's voice. It does **not** reprint material from Weiss / Stark / Enns / other consultancy authors — those books still belong to their authors. The skills capture process and patterns; brand language stays with its owners.
